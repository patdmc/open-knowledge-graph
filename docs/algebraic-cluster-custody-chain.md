## Grades

**A** -- definitional: the mapping holds under translation with no interpretive gap.

## Structure of the Algebraic Cluster

The four sources form a natural hierarchy:

```
Kac (1990): Kac-Moody algebras and their Dynkin diagrams
    ↓
FLM (1988): Vertex operator algebras (VOAs) — the conformal field theory analog
    ↓
Borcherds (1992): Moonshine proof — Monster group as automorphism group of V^natural
    ↓
Buhl-Karaali (2008): Spanning theorems for Möbius VAs — generalization to arbitrary N
```

This hierarchy matters for the custody chain: propositions at the Borcherds/FLM level build on (and assume) results at the Kac level. When evaluating independence from UB, the relevant question is whether the full derivation chain (Kac → FLM → Borcherds → Buhl-Karaali) is independent of the UB framework. It is: none of these papers references bounded rationality, knowledge graphs, or the UB framework.

## KM1: Dynkin Diagram Generates Infinite Kac-Moody Algebra

**Specific to**: Kac (1990), Chapter 1 (generalized Cartan matrix, Dynkin diagram, generators and relations construction).

**Maps to**: NV01 (graph necessity), EC05 (incompleteness)
**Emergent intersection**: EM16 (with GB0)

### Evidence Grade: B+ (structural, near definitional)

The correspondence is: the Dynkin diagram (a finite graph) generates the full infinite-dimensional Kac-Moody algebra via generators and relations. Under translation:

```
Dynkin diagram = K/F graph (minimal finite structure)
Generators (nodes) = K_i dimensions
Edges (interaction coefficients) = relational structure of F
Full algebra = the infinite content of W generated by K/F
```

The translation requires identifying "generator of an algebra" with "dimension of knowledge." This is one step of interpretation: both are primitive elements from which a larger structure is built, but generators are formal algebraic objects and knowledge dimensions are epistemic. The step is well-motivated by the structural parallel but is not trivial.

The ADE classification (A_n, D_n, E_6, E_7, E_8 for finite-dimensional Lie algebras) provides a taxonomy of all possible minimal graph structures. This has a specific implication for the UB framework: if K/F graphs are classifiable by the same ADE taxonomy, then the possible knowledge structures for any entity at a given capacity C_n form a known finite list. This is a strong structural prediction that neither Kac nor NV01 makes independently.

### Independence

Complete. Kac's derivation is pure mathematics: generators and relations, root systems, Weyl groups. No epistemic motivation.

### Summary

| Criterion | Assessment |
|---|---|
| Evidence grade | B+ |
| Translation step | "Generator" ≡ "knowledge dimension" (one step, well-motivated) |
| Unique contribution | ADE taxonomy as classification of K/F graph types |
| Derivation path | Fully explicit in Kac Ch. 1 |

## KM2: Real vs. Imaginary Root Partition

**Maps to**: NV02 (M/N bound), EC05 (incompleteness)

### Evidence Grade: B (structural)

Real roots (finite multiplicity) and imaginary roots (infinite multiplicity) partition the root system of an infinite-dimensional Kac-Moody algebra. The correspondence:

```
Real roots (finite multiplicity) = M: dimensions the entity has represented
Imaginary roots (infinite multiplicity) = N - M: dimensions beyond current context
M < N always because the imaginary root sector is always larger
```

The structural parallel is exact: in both frameworks, there is a partition of the total structure into a finite "represented" part and an infinite "unrepresented" part. The M < N inequality maps to the fact that the imaginary root sector is always infinite (for indefinite Cartan matrix), even as M grows.

The interpretive step: "root of a Lie algebra" and "dimension of W" must be identified. Root systems are specific algebraic objects (elements of a dual vector space satisfying reflection conditions). World dimensions are epistemic objects. The structural parallel holds, but the objects are different.

### Summary

| Criterion | Assessment |
|---|---|
| Evidence grade | B |
| Translation step | "Root" ≡ "world dimension" |
| Status | Sound structural observation |

## KM3: Weyl Group as Compression Symmetry

**Maps to**: EC08 (compression factoring), NV01 (graph necessity)

### Evidence Grade: B (structural)

The Weyl group generates an infinite orbit from a finite fundamental domain. Compressing the full root system to the fundamental domain is the exact operation EC08 describes: the minimal factored representation that carries all information about the full structure.

```
Weyl group action = the symmetry group of the compression
Fundamental domain = the K/F graph (minimal factored representation)
Full root system = the full K (before compression)
Weyl orbit = all equivalent representations (related by symmetry)
```

The step: "fundamental domain under Weyl group" and "K/F graph" must be identified as the same abstract concept: a minimal object that carries all information about the full structure up to symmetry. Well-motivated: both are minimal representatives of infinite equivalence classes under their respective symmetry groups.

### Summary

| Criterion | Assessment |
|---|---|
| Evidence grade | B |
| Unique contribution | Compression has a symmetry group (the Weyl group) |

## KM4: Character Formula and Modular Forms

**Maps to**: EC08 (compression factoring)

### Evidence Grade: C (analogical)

Kac's character formula encodes the representation theory of Kac-Moody algebras in modular forms -- generating functions that encode infinite arithmetic data in a finite formula with modular symmetry. The correspondence: modular forms are maximally compressed; they carry infinite information in a finite analytic expression.

The gap: "modular form" and "K/F graph" are not obviously the same object. A modular form is an analytic function with specific transformation properties under SL(2,Z). A K/F graph is a combinatorial structure. The shared property (both are maximally compressed representations of infinite content) is real but does not make them structurally equivalent objects.

The correspondence is most useful as an example of EC08 compression taken to an extreme, not as an independent derivation.

### Summary

| Criterion | Assessment |
|---|---|
| Evidence grade | C |
| Status | Illustrative example of EC08; not an independent derivation |

## BO1: Locality Axiom as Provenance Independence

**Specific to**: Borcherds (1992), Def. 1 (vertex algebra axioms).

**Maps to**: NV04 (intersection acceleration), D13 (provenance)
**Emergent intersection**: EM17

### Evidence Grade: B+ (structural, near definitional)

The locality axiom: (z - w)^n [Y(a,z), Y(b,w)] = 0 for large enough n. Vertex operators commute at separated points. The translation:

```
Y(a,z) = K_i (knowledge-generating operation)
Y(b,w) = K_j
(z-w)^n [Y(a,z), Y(b,w)] = 0 = K_i and K_j have independent provenance
OPE Y(a,z)Y(b,w) = K_collective from NV04 intersection
OPE coefficient = |K_j \ K_i| new dimensions produced
```

Under this translation, the locality axiom is precisely the independence condition for NV04. The formal requirement "(z-w)^n kills the commutator" maps to "the provenance chains of K_i and K_j do not share sources."

The step that prevents grade A: "vertex operator commutativity at separated points" and "provenance independence of derivation" must be identified. Both express independence of two entities, but commutativity is an algebraic property (operator reordering) while provenance independence is an epistemic property (no circular sourcing). The step requires accepting that algebraic commutativity of operators at separated points and epistemic independence of derivation chains are the same abstract condition. This is the key structural claim of EM17 and it is well-motivated: commutativity in the OPE sense measures whether the two operators can be applied in either order (= whether their sources are independent), which is the same condition as provenance independence.

**Why this is important for D13**: the locality axiom makes provenance independence a *formal consistency requirement*, not just a bookkeeping convention. A vertex algebra with non-local operators (that don't commute at separated points) is inconsistent as an algebraic structure. This means: a K_collective formed from K_i and K_j with circular provenance is algebraically inconsistent. The vertex algebra framework turns the D13 provenance requirement from a convention into a theorem.

### Derivation Path (EM17)

Step 1: Vertex operators Y(a,z) and Y(b,w) must commute at separated points (locality axiom).
Step 2: Commutativity at separation = independence of operators at those points.
Step 3: Operator independence = the derivation of a and b do not share sources (provenance independence).
Step 4: NV04 requires K_i and K_j to be independently derived for the intersection to produce valid new content.
Step 5: Locality axiom ≡ NV04 independence condition.
Step 6: OPE = the formal intersection rule; OPE coefficient = |K_j \ K_i|.

Step 3 is the interpretive step: algebraic independence → epistemic provenance independence. Well-motivated but one step.

### Summary

| Criterion | Assessment |
|---|---|
| Evidence grade | B+ |
| Translation step | Operator commutativity ≡ provenance independence |
| Derivation path | 6 steps explicit |
| Unique contribution | Provenance independence is algebraically necessary (consistency condition) |
| C_1 contribution | NV04 grounding; OPE as formal intersection rule |

## BO2: Monster Lie Algebra and Monstrous Moonshine

**Maps to**: EC08 (compression), EM09 (collective intelligence)

### Evidence Grade: B (structural)

The moonshine correspondence (Monster group encoded in coefficients of the j-function) is an extreme instance of EC08: the most complex discrete symmetry (Monster, order ~ 8 x 10^53) is exactly captured by the most symmetric analytic object (the j-function, a modular form of weight 0). The correspondence maps to: maximum complexity under compression = maximum symmetry.

This is the same claim as EM22 (central charge c = C_n, Moonshine module as maximum-compression structure) but viewed from the Lie algebra perspective rather than the VOA perspective. The convergence of BO2 and EM22 on the same structural point (Monster = symmetry of maximum compression at c = 24) strengthens both.

### Summary

| Criterion | Assessment |

| Evidence grade | B |
| Relation to EM22 | Same structural claim from different angle |

## BO3: Borcherds Identity as Compressed Intersection Grammar

**Maps to**: EC08 (compression factoring), NV01 (graph necessity)

### Evidence Grade: B (structural)

The Borcherds identity is the single relation encoding the full vertex algebra structure: it generalizes both the Lie algebra Jacobi identity and the OPE associativity into one identity. Under translation: one identity encodes all valid intersections (all valid K_collective constructions). This is EC08 at maximum: the compressed sufficient statistic for the complete intersection grammar.

The step: "the Borcherds identity encodes all operator relations" and "the K/F graph encodes all knowledge structure" must be identified. Both are single finite objects encoding infinite relational content, but the objects are different (a formal identity vs. a graph). The structural parallel is clear; the objects are not the same type.

## BO4: Translation Covariance (T|0> = 0)

**Maps to**: EC02 (improvement rate), NV02 (M/N bound)

### Evidence Grade: B (structural)

[T, Y(a,z)] = d/dz Y(a,z) means the translation operator T acts as differentiation. T|0> = 0 means the vacuum state does not evolve under translation. The correspondence: T|0> = 0 = A = 0 (the entity that does not act does not learn; the vacuum does not advance). The differentiation structure means T advances the entity along the dimension z, measuring the rate of change -- the eta_M rate of learning.

The step: "vacuum state" and "A = 0 agent" must be identified. Both are zero-action ground states. Well-motivated.

## FL1: Central Charge as Capacity Measure

**Specific to**: FLM (1988), Chapter 1.

**Maps to**: EC01 (context bound), EC08 (compression)
**Emergent intersection**: EM22

### Evidence Grade: B+ (structural)

The Virasoro algebra central charge c measures the conformal degrees of freedom of the VOA. Translation:

```
Central charge c = C_n (context capacity)
VOA with c = capacity-c system
Virasoro algebra = the algebra of context-constrained operations
```

The identification c = C_n is clean under EM04 (C_n = Shannon channel capacity). EM04 establishes C_n as a physical information capacity. FL1 establishes c as the conformal information capacity. Both measure the maximum information representable in a bounded conformal/contextual system.

The step: "conformal degrees of freedom" and "context capacity" must be identified. Both measure the number of independent variables a system can represent, in conformal and epistemic senses respectively. The conformal degrees of freedom are more specific (they measure the response to conformal transformations), but in the VOA setting they quantify the information capacity of the system under conformal dynamics. The identification requires accepting that conformal dynamics and epistemic context-boundedness are the same constraint in different formalisms.

### Summary

| Criterion | Assessment |

| Evidence grade | B+ |
| Translation step | Conformal degrees of freedom ≡ context capacity |
| Unique contribution | c = C_n gives the capacity bound a precise algebraic measure |

## FL2: Moonshine Module as Maximum-Compression Structure

**Maps to**: EC08 (compression), EM09 (collective intelligence)
**Part of**: EM22

### Evidence Grade: B (structural)

V^natural (the Moonshine module) is the unique VOA with c = 24, no weight-1 states, and Monster automorphism group. Under translation: at C_n = 24, the unique maximum-entropy, maximum-compression knowledge structure has Monster symmetry.

The step: "automorphism group of the VOA" and "symmetry group of the optimal K/F graph" must be identified. Both describe the symmetry of the optimal compression structure at a given capacity. The identification is well-motivated: automorphisms of V^natural are the symmetries that preserve all its algebraic structure, just as symmetries of the optimal K/F graph would be those that preserve all relational information.

The Moonshine consequence is the most structurally surprising result in the algebraic cluster: the largest sporadic simple group (Monster, order ~ 8 x 10^53) is the symmetry of the optimal knowledge structure at C_n = 24. This is not expected from either the VOA theory or the UB framework taken alone.

### Summary

| Criterion | Assessment |
|---|---|
| Evidence grade | B |
| Unique contribution | Optimal compression at specific capacity has the Monster symmetry |
| Structural surprise | Monster group appears as symmetry of optimal K/F graph at C_n = 24 |

## FL3: OPE as Formal Intersection Rule

**Maps to**: NV04 (intersection), EC07 (collective knowledge)

### Evidence Grade: B (structural)

The OPE Y(a,z)Y(b,w) gives the full expansion of two vertex operators interacting. The leading singular terms give the "most emergent" content (lowest powers of (z-w)); regular terms give the "background" content. Translation: the OPE is the formal intersection rule for K_i and K_j. The singular OPE coefficient = the |K_j \ K_i| new dimensions.

The step: "singular OPE coefficients" and "new knowledge dimensions from NV04" must be identified. The singular terms in the OPE arise precisely from the non-trivial interaction between a and b -- they are zero if a and b are "trivially related." This maps to: the new dimensions in NV04 arise precisely from the non-trivial difference K_j \ K_i (they are zero if K_i ⊇ K_j). Structurally sound.

### Summary

| Criterion | Assessment |
|---|---|
| Evidence grade | B |
| Unique contribution | OPE is the formal intersection rule NV04 lacks |

### Cluster Pattern

The algebraic cluster has no grade-A results (unlike CL4 in compiler theory). The cluster's strength is breadth and coherence: twelve propositions across four papers all map to the same structural region of the UB framework (NV01, NV04, EC08, EC01). No single mapping is definitional, but the convergence of twelve independent structural observations from four different subfields of algebra on the same framework nodes constitutes strong aggregate evidence.

The two B+ results (KM1 and BO1) are the anchor points. KM1 anchors the graph necessity cluster (EM16). BO1 anchors the provenance independence cluster (EM17). EM22 (central charge = C_n) sits between them, with FL1 and FL2 at grade B.

### What the Algebraic Cluster Provides That No Other Cluster Does

1. **A taxonomy of K/F graph types** (Dynkin diagram ADE classification). No other cluster provides a finite enumeration of all possible minimal graph structures.

2. **A formal consistency requirement for provenance independence** (locality axiom). No other cluster makes provenance independence into an algebraic necessity rather than a convention.

3. **A unique maximum-compression structure at a specific capacity** (Moonshine module at c = 24). No other cluster identifies the optimal knowledge structure at a particular capacity and names its symmetry group.

4. **A complete formal intersection grammar** (OPE + Borcherds identity). No other cluster provides an explicit formula for what K_collective is produced by each K_i x K_j pair.

These four unique contributions make the algebraic cluster the most formally complete realization of the UB core structure in the corpus.
